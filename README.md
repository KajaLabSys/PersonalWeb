# KajaLab osobní web

Jednoduchý statický osobní web pro Karla Jandu.

## Lokální spuštění

```bash
npm install
npm run dev
```

Pak otevřít adresu, kterou vypíše `vite`, obvykle `http://localhost:5173`.

## Nasazení přes Coolify

Projekt je připravený pro nasazení z `Dockerfile`.

Základní nastavení:

- typ aplikace: Dockerfile / statický web
- interní port: `80`
- doména: `kajalab.cz`
- build context: kořen repozitáře
